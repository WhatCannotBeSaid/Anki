# Anki-Theme

精美的 Anki 自用主题模板，包含“单词”和“填空”两种类型，旨在提升学习体验和视觉美感。

## 🌟 模板特色

### 1. 单词模板 (Word Template)
专门为英语单词背诵设计，具有以下特点：
- **现代化视觉设计**：大字体、优雅的排版，支持日间/夜间模式自动切换。
- **交互式发音**：点击单词即可播放发音。
- **悬挂式词性展示**：自动识别并将词性（如 n., v., adj. 等）悬挂在释义左侧，整齐美观。
- **智能例句显示**：默认显示英文例句，鼠标悬停或点击即可查看中文翻译。

**基础字段需求**：
- `单词`：单词原文
- `音标`：单词音标
- `发音`：[sound:filename.mp3]
- `释义`：中文释义（支持多行，支持常见词性前缀）
- `英文例句`：句子内容
- `中文例句`：句子的中文翻译

### 2. 填空模板 (Cloze Template)
适用于学术阅读、长句记忆和短语填空：
- **学术风格排版**：类似于论文阅读器的配色与边框，专注度更高。
- **彩色词性胶囊**：定义区域自动将不同词性（动词、名词、形容词等）标记为彩色胶囊。
- **悬浮翻译气泡**：英文释义中包含括号内的中文翻译时，会自动转换为悬浮气泡，只有鼠标移入时才显示，减少视觉干扰。
- **笔记便签**：带有醒目边栏的 Notes 区域，适合记录额外知识点。

**基础字段需求**：
- `Source`：来源（如文章名、书籍章节）
- `Text`：包含 {{c1::cloze}} 的句子主体
- `Definition`：定义（格式如：`n. definition (中文释义)`）
- `Notes`：补充笔记

## 🚀 如何使用

1. 在 Anki 中新建或修改卡片类型。
2. 将对应文件夹（`单词` 或 `填空`）中的 `.html` 内容分别复制到 Anki 的“正面模板”和“背面模板”中。
3. 将对应的 `.css` 内容复制到 Anki 的“样式”区域。
4. 确保你的字段名称与上述“基础字段需求”匹配，或者手动修改模板中的 `{{字段名}}`。
